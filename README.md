# kilo

A terminal text editor.


## How to use?

### Build from source

After cloning the repo, run

```
cargo build
```

This will create an executable at `./target/debug/kilo`.

### Usage

Open the text editor

```
./target/debug/kilo
```

Open the text editor with a file

```
./target/debug/kilo <location_of_your_file>
```
